## Списки

Для создания ненумерованного списка необходимо пункты выделить символом (*). 

Например, вот так
* Элемент 1
* Элемент 2
* Элемент 3

Для создания нумерованного списка необходимо пронумеровать строки. 

Например,вот так
1. Элемент 1
2. Элемент 2
